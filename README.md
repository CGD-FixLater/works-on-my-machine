# configs

我的配置与折腾记录合集 —— 每搞定一套环境/工具的配置，就整理成「踩坑笔记 + 一键脚本」放进来，方便换机器复用，也分享给可能踩同样坑的人。

每个主题一个子文件夹，通常包含：
- 一份 Markdown 记录：讲原理、分步操作、踩过的坑、回滚方法。
- 一个一键脚本：想偷懒可直接跑。

## 目录

| 配置 | 适用环境 | 作用 |
|------|----------|------|
| [rime-ice-setup](./rime-ice-setup/) | Ubuntu 22.04 / GNOME / Wayland | 把 ibus 默认拼音换成 **雾凇拼音 (rime-ice)**，获得整句输入 + 大词库 + 智能联想 |

> 以后每加一套配置，在上表追加一行即可。

## 使用

进对应文件夹，按里面的 `README` 或 `*.md` 操作；多数主题也提供一键脚本，例如：

```bash
cd rime-ice-setup
bash install-rime-ice.sh
```
